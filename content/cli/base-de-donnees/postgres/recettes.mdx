## Filtrer par un tableau de jsonb

```sql
with flattened_query as (
  select id, unnest(my_array_col) as "my_jsonb_col" from my_table
)

select id, my_jsonb_col from flattened_query where my_jsonb_col->'key' = 'my value';
```

## Pattern Matching

### Filter par nombres decimaux

```sql
select * from table where number ~ '(^\d+(\.\d{1,2}))';
```
