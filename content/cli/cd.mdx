title: cd
---

La navigation d’un répertoire à un autre s’effectue avec la commande `cd` succédée du nom du répertoire.

<C i={`cd /`}>Permet de se retrouver à la racine du disque.</C>

<C i={`cd`}>Accéder directement au répertoire de l’utilisateur.</C>

<C i={`cd /var/www/`}>Aller dans le répertoire `/var/www`.</C>

<C i={`cd ..`}>
  Remonter dans le répertoire parent à partir de là où vous êtes.
</C>

<C i={`cd -`}>Permet de revenir au répertoire précédent.</C>

<C i={`pwd`}>
  Renvoyer le chemin absolu du répertoire courant ce qui est utile puisqu’en
  général le shell n’affiche que le nom du répertoire courant.
</C>
