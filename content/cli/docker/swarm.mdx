<C i={`docker swarm init`}>Initialise l'hote en mode swarm, par defaut ce noeud devient leader.</C>

<C i={`docker node ls`}>Affiche la liste de tous les noeuds rattache à ce swarm.</C>

<C i={`docker stack deploy -c compose.yml name_of_stack`}>Demarre une stack à partir d'un fichier docker-compose.</C>
