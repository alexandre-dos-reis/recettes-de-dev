Vim est un éditeur de texte, c’est-à-dire un logiciel permettant la manipulation de fichiers texte. Il est directement inspiré de vi dont il est le clone le plus populaire. Chaque commande est préfixé par le mode auquel il appartient.

<C i={`vi{`} env={`[NORMAL]`}>Sélectionne tous le texte contenu dans une paire d'accolade, très utile pour séléctionner rapidement du code.</C>

<C i={`vi(`} env={`[NORMAL]`}>Sélectionne tous le texte contenu dans une paire de parenthèse.</C>

<C i={`vi[`} env={`[NORMAL]`}>Sélectionne tous le texte contenu dans une paire de crochets.</C>
