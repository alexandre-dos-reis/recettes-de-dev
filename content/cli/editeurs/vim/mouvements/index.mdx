Les mouvements Vim sont à différencier de l'éditeur Vim. Il existe pour tous les IDEs modernes des extensions Vim qui permetent d'obtenir les mêmes mouvements que dans l'éditeur Vim.

Voici l'anatomie d'un mouvement vim: `COMMAND` `COUNT` `ACTION`
