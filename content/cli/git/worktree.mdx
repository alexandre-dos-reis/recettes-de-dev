## Mise en place d'un depot worktree.

```
mkdir project && cd project
git clone --bare git@github.com:user/project.git .bare
echo "gitdir: ./.bare" > .git
```

## Commandes usuelles

<C i={`git worktree add <branche> <dossier>`}>
    Ajouter une branche au worktree.
</C>

<C i={`git worktree add -b <feature-branch> <branche-cible> <dossier>`}>
    Creer une nouvelle branche au worktree depuis une branche cible en precisant le chemin du dossier.
</C>
