La commande `man` permet d'afficher la documentation relative à une commande.

<C i={`man cd`}>Affiche les informations sur la commande 'cd'.</C>

<C i={`man -u`}>Rafraîchie la base de donnée du man.</C>

<C i={`man -f`}>
  Affiche, si elle est disponible, une courte description de la page de manuel
  de la commande.
</C>
