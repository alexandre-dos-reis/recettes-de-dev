Node.js est une plateforme logicielle libre en JavaScript orientée vers les applications réseau événementielles hautement concurrentes qui doivent pouvoir monter en charge. Elle utilise la machine virtuelle V8, la librairie libuv pour sa boucle d'évènements, et implémente sous licence MIT les spécifications CommonJS. NPM est son gestionnaire de paquets.

<C i={`node app.js`}>Exécute le fichier `app.js`.</C>

<C i={`npm init`}>Initialise un nouveau projet Node avec la création d'un fichier `package.json` qui répertorie la liste des dépendances. </C>

<C i={`npm install`}>Installe toutes les dépendances présentes dans le fichier `package.json`.</C>

<C i={`npm install dotenv`}>Installe le paquet `dotenv` et le rajoute au fichier des dépendances `package.json`.</C>

<C i={`npm install -g nodemon`}>Installe le paquet `nodemon` de manière globale. A la prochaine installation du paquet, `npm` ne le téléchargera pas puisqu'il est installé sur le système.</C>

<C i={`npm update`}>Met à jour les paquets en production.</C>

<C i={`npm update lodash`}>Met à jour le paquet `lodash`.</C>

<C i={`npm uninstall mysql`}>Désinstalle le paquet `mysql`.</C>

<C i={`npx http-server`}>Permet d'éxecuter le paquet `http-server` sans le télécharger et l'installer, utile pour le test.</C>

<C i={`npm run start`}>Exécute la commande `start` qui est contenue dans le fichier package.json.</C>
