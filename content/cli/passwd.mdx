title: passwd
---

Permet de changer le mot de passe d'un utilisateur.

<C i={`/etc/shadow`}>Fichier contenant les mots de passe des utilisateurs.</C>

<C i={`passwd`}>Change le mot de passe de l'utilisateur courant.</C>

<C i={`passwd -Sa`}>
  Montre le statut du mot de passe pour tous les utilisateurs.
</C>

<C i={`passwd John`}>Change le mot de passe de l'utilisateur `John`.</C>

<C i={`passwd -d John`}>
  Efface le mot de passe de l'utilisateur `John`, le compte sera alors sans mot
  de passe.
</C>

<C i={`passwd -e John`}>
  Fait expirer le mot de passe de `John`, cela a pour effet de forcer le
  changement de mot de passe Ã  sa prochaine connexion.
</C>
