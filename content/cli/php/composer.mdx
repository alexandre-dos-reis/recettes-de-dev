**Composer** est le gestionnaire de paquets pour PHP inspiré de Node. Il permet aussi l'import automatique de classes grâce à son autoloader. Le catalogue est consultable sur  [packagist.org](https://packagist.org/).

<C i={`composer require guzzlehttp/guzzle`}>Permet d'installer le paquet `guzzle`, l'installe dans un dossier nommé `Vendor` qui est le dossier par défaut des installations de librairies.</C>

<C i={`composer remove guzzlehttp/guzzle`}>Désinstalle le paquet `Guzzle`.</C>
