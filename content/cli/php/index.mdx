PHP peut aussi s'utiliser en ligne de commande pour éxecuter des fichiers dans le terminal ou pour démarrer un serveur.

<C i={`php -S localhost:8000 -t public`}>Démarre un serveur PHP disponible sur le port 8000 à partir du dossier public, le fichier index.php sera chargé par défaut.</C>

<C i={`php -S localhost:8000 -c php.ini`}>Démarre un serveur PHP disponible sur le port 8000 en chargeant un fichier de configuration php différent.</C>

<C i={`php -S www.exemple.com:8000`}>Démarre un serveur PHP disponible à cette adresse, mais il faudra la renseigner dans le fichier hosts.</C>
