title: uname
---

Affiche les informations sur la machine et l'OS.

<C i={`uname`}>Affiche le nom du kernel.</C>

<C i={`uname -s`}>idem</C>

<C i={`cat /etc/debian_version`}>Affiche la version de Debian.</C>

<C i={`uname -m`}>Affiche le type d'architecture.</C>

<C i={`uname -n`}>Affiche le nom d'hôte du réseau.</C>

<C i={`uname -p`}>Affiche le type du processeur.</C>

<C i={`uname -v`}>Affiche la version du kernel.</C>

<C i={`uname -r`}>Affiche la release du kernel.</C>

<C i={`uname -i`}>Affiche la plateforme d'environnement.</C>

<C i={`uname -o`}>Affiche le système d'exploitation.</C>

<C i={`uname -a`}>Affiche toutes les informations disponibles.</C>
