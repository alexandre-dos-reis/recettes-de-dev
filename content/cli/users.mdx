Relatif à la gestion des utilisateurs.

<C i={`/etc/passwd`}>Fichier contenant les utilisateurs du système.</C>

<C i={`id`}>
  Permet de connaître les informations essentielles sur un utilisateur : uid,
  gid, groupes secondaires
</C>

<C i={`whoami`}>Renvoie le login de l'utilisateur courant</C>

<C i={`adduser`}>
  Script interactif écrit en Perl qui utilise l'exécutable binaire useradd et
  qui permet de créer un utilisateur complet. User Friendly !
</C>

<C i={`useradd`}>
  Permet de créer un utilisateur mais il faut spécifier son home, shell avec les
  options.
</C>

<C i={`userdel -r alex`}>
  Supprime un utilisateur, l’option -r permet de supprimer le home. Les fichiers
  présent dans le système seront à supprimer manuellement.
</C>
