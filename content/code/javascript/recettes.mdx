## Ne pas utiliser de boucle foreach avec les promesses.

A ne jamais faire !

```js
items.foreach(async (i) => {
  await i.someAsyncFunc();
});
```

Mais plutÃ´t :

```js
await Promise.all(items.map(async (i) => {
    await i.someAsyncFunc();
  }));
```
